# Waic Deckbuilder

## 開発環境

- Node.js v24.3.0
- pnpm v10.12.4

## インストールとセットアップ

1.  **リポジトリをクローンする**:

    ```bash
    git clone https://github.com/suika-lunch/waic-deckbuilder.git
    cd waic-deckbuilder
    ```

2.  **依存関係をインストールする**:

    ```bash
    pnpm i
    ```

3.  **開発サーバーを起動する**:
    ```bash
    pnpm dev
    ```

## 技術スタック

- Vue.js
- Vite
- TypeScript
- Tailwind CSS
- Pinia
